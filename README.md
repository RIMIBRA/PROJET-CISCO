# Network Engineering Projects — CCNA 3 & 4

## Overview
Practical Cisco networking projects completed as part of 
CCNA 3 and CCNA 4 training at IST Ouagadougou, Burkina Faso.

## Projects Included

### 1. CCNA 3 — Advanced Routing & Switching
- OSPF configuration
- Inter-VLAN routing
- Spanning Tree Protocol (STP)

### 2. CCNA 4 — WAN Technologies
- NAT and ACL configuration (`nat et acl.pkt`)
- WAN link setup
- Network security basics

### 3. Router on USB Key
- Portable router configuration lab

## Tools Used
- Cisco Packet Tracer

## About
These projects reflect hands-on network engineering practice
and form the technical foundation for my research interests
in network security and cybersecurity infrastructure.
